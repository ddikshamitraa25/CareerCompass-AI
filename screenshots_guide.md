# 📸 Screenshots Guide — Career Compass AI

For a complete and polished submission (and for the README's Screenshots
section), capture the following screenshots after running the app locally
with `streamlit run app.py`.

## Required Screenshots

1. **Home Page (Hero + Feature Overview)**
   - Show the full hero banner, welcome cards, and feature badge list.
   - File suggestion: `screenshots/01_home.png`

2. **Sidebar Navigation (Expanded)**
   - Capture the sidebar showing all 10 features with icons and the API
     status indicator.
   - File suggestion: `screenshots/02_sidebar.png`

3. **Explain Career — Input + Response**
   - Enter a career (e.g., "Data Scientist") and capture the generated
     explanation inside the styled response box.
   - File suggestion: `screenshots/03_explain_career.png`

4. **Learning Roadmap — Full Output**
   - Generate a roadmap and screenshot the phased breakdown.
   - File suggestion: `screenshots/04_learning_roadmap.png`

5. **Daily Study Plan — With Hours Selector**
   - Show the dropdown for hours/day and the resulting weekly plan.
   - File suggestion: `screenshots/05_daily_study_plan.png`

6. **Interview Questions — With Experience Level**
   - Show the experience-level dropdown and the 10 generated questions.
   - File suggestion: `screenshots/06_interview_questions.png`

7. **Generate Quiz — Multiple Choice Output**
   - Capture a generated quiz with visible options A–D and answers.
   - File suggestion: `screenshots/07_generate_quiz.png`

8. **Ask Anything — Free-Form Question**
   - Show a custom question typed by the user and the AI's answer.
   - File suggestion: `screenshots/08_ask_anything.png`

9. **Motivation Quote — Mood Selector**
   - Capture the mood dropdown and the resulting quote card.
   - File suggestion: `screenshots/09_motivation_quote.png`

10. **Loading Spinner (In Action)**
    - Quickly screenshot the app *while* a request is processing to show
      the "🧠 Thinking..." spinner.
    - File suggestion: `screenshots/10_loading_spinner.png`

11. **Error Handling Example**
    - Temporarily remove/rename your `.env` file, restart the app, and
      screenshot the friendly "API key not found" warning message and the
      🔴 API status indicator in the sidebar.
    - File suggestion: `screenshots/11_error_handling.png`

12. **About Page**
    - Capture the About section showing project info, tech stack, and
      version/license.
    - File suggestion: `screenshots/12_about.png`

## Tips for Great Screenshots

- Use a browser window at least **1400px wide** for a clean, uncropped
  desktop layout.
- Use light-content examples (a real career name, a real question) rather
  than placeholder text like "test" or "asdf" — it looks more professional.
- Keep the browser's own UI (tabs, bookmarks bar) out of frame if possible.
- Save all screenshots inside a `screenshots/` folder at the project root so
  they can be referenced directly in `README.md`.
